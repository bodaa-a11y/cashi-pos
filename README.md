# نظام كاشي لإدارة نقاط البيع (Cashi POS)

نظام كاشير متكامل وحديث للمطاعم والوجبات السريعة مبني باستخدام تقنيات الويب الحديثة (React + Express) ومحزّم كبرنامج سطح مكتب باستخدام Electron مع دعم كامل للغة العربية والطباعة الحرارية ومواصفات الفاتورة الضريبية لهيئة الزكاة والضريبة والجمارك (ZATCA).

---

## ✨ المميزات الرئيسية
* **واجهة مستخدم POS سريعة وتفاعلية:** مبنية باستخدام React و Tailwind CSS.
* **تطبيق سطح مكتب (Electron):** يعمل كبرنامج مستقل على نظام تشغيل Windows.
* **سيرفر Express مدمج:** يعمل محلياً لإدارة العمليات بشكل مستقل وسريع.
* **طباعة الفواتير وتذاكر المطبخ:** دعم كامل للطابعات الحرارية بعرض 80mm وإلغاء الهوامش تلقائياً.
* **الفاتورة الضريبية المبسطة (ZATCA):** توليد تلقائي للـ QR Code المشفر والمطابق لتعليمات هيئة الزكاة والضريبة والجمارك.
* **إدارة المستودع والمخزون:** تعيين رصيد المواد الخام وتحديد استهلاك كل صنف بالمنيو (الوصفات) للخصم التلقائي عند البيع.
* **لوحة مراقبة المدير عن بعد (Remote Manager Dashboard):** لوحة تحكم ذكية لعرض المبيعات والتقارير المالية والورديات عن بعد عبر الشبكة المحلية.

---

## 💻 متطلبات النظام
* نظام تشغيل **Windows 10 / 11** (للبرنامج المحزّم).
* بيئة تشغيل **Node.js** (الإصدار 18 أو أحدث) للتطوير والبناء.

---

## 🛠️ خطوات التشغيل والتطوير (للمطورين)

### 1. تثبيت الحزم والمكتبات
قم بتثبيت الحزم المطلوبة من المجلد الرئيسي للمشروع:
```bash
npm install
```

### 2. تشغيل السيرفر والواجهة الأمامية في بيئة التطوير
لتشغيل المشروع بالكامل في بيئة التطوير (React + Express + Electron):
```bash
npm run electron:dev
```
سيقوم هذا الأمر ببناء الواجهة الأمامية وتشغيل خادم Express الداخلي ثم تشغيل برنامج Electron لسطح المكتب.

### 3. بناء وتجميع حزمة التثبيت النهائية (Build)
لإنشاء ملف التثبيت المستقل لويندوز (`Cashi-Setup-1.0.0.exe`):
```bash
npm run electron:build
```
سيتم حفظ ملف التثبيت النهائي داخل مجلد `release/`.

---

## 📁 هيكلية قاعدة البيانات المحلية
* يتم حفظ كافة البيانات محلياً داخل ملف `db.json` في مسار بيانات التطبيق الخاص بـ Electron:
  `%APPDATA%/كاشي - نظام الكاشير المتكامل/db.json`

## 🛡️ الأمان والتحقق
* جميع الـ API محمية بالكامل بميدلوير تحقق ذكي (`JWT-HMAC`).
* يتم التحقق من إجماليات الفاتورة وأسعار الأصناف على السيرفر قبل إتمام البيع لمنع أي تلاعب مالي.
* يدعم حظر محاولات التخمين المتكرر للـ PIN للموظفين (قفل مؤقت 60 ثانية بعد 5 محاولات خاطئة).
